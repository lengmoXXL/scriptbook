# Markdown 语法演示

本文档展示 Scriptbook 支持的 Markdown 语法和渲染效果。

## 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

渲染效果：

# 一级标题
## 二级标题
### 三级标题

---

## 文本格式

```markdown
**粗体文本**
*斜体文本*
***粗斜体文本***
~~删除线文本~~
`行内代码`
```

渲染效果：

**粗体文本**
*斜体文本*
***粗斜体文本***
~~删除线文本~~
`行内代码`

---

## 列表

### 无序列表

```markdown
- 项目一
- 项目二
  - 子项目二之一
  - 子项目二之二
- 项目三
```

渲染效果：

- 项目一
- 项目二
    - 子项目二之一
    - 子项目二之二
- 项目三

### 有序列表

```markdown
1. 第一步
2. 第二步
3. 第三步
```

渲染效果：

1. 第一步
2. 第二步
3. 第三步

### 任务列表

```markdown
- [ ] 未完成任务
- [x] 已完成任务
```

渲染效果：

- [ ] 未完成任务
- [x] 已完成任务

---

## 链接和图片

```markdown
[百度](https://www.baidu.com)

![替代文本](https://via.placeholder.com/150)
```

渲染效果：

[百度](https://www.baidu.com)

---

## 引用块

```markdown
> 这是一段引用内容。
> 可以跨越多行。

> **带格式的引用**
> - 支持列表
> - 支持其他元素
```

渲染效果：

> 这是一段引用内容。
> 可以跨越多行。

> **带格式的引用**
> - 支持列表
> - 支持其他元素

---

## 代码块

### 普通代码块

```bash
echo "Hello World"
```

### 带语言标注

```python
def hello():
    print("Hello, World!")
```

```javascript
console.log("Hello, World!");
```

```json
{
  "name": "Scriptbook",
  "version": "1.0.0"
}
```

---

## 表格

```markdown
| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |
```

渲染效果：

| 表头1 | 表头2 | 表头3 |
|-------|-------|-------|
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

### 对齐表格

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|------:|
| 内容1 | 内容2 | 内容3 |
```

渲染效果：

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|------:|
| 内容1 | 内容2 | 内容3 |

## 内联 HTML

```html
<kbd>Ctrl</kbd> + <kbd>C</kbd> 复制

<mark>高亮文本</mark>
```

渲染效果：

<kbd>Ctrl</kbd> + <kbd>C</kbd> 复制

<mark>高亮文本</mark>

---

## 脚注

```markdown
这是一段包含脚注的文字。[^1]

[^1]: 这是脚注内容。
```

---

> **提示**：Scriptbook 还支持在代码块中嵌入可执行脚本，点击右侧的 ▶ 按钮即可运行。
